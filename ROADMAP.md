# 🗺️ AURA Product Roadmap

Welcome to the official roadmap for **AURA (Artificial Unified Reasoning Assistant)**. 
Our vision is to build the fastest, most capable hybrid intelligence system that seamlessly bridges cloud-scale reasoning with secure, localized tool execution.

---

## 🎯 Phase 1: Foundation (Current Phase)
*The core architecture and initial release.*

- [x] **Glassmorphic Flutter Client**: Build the beautiful, transparent desktop overlay and chat interfaces.
- [x] **Unified Gateway Pattern**: Implement the Bun/FastAPI backend router for sub-millisecond response times.
- [x] **Hybrid Inference Core**: Connect to Groq (Llama-3.3-70B) for speed and local OpenVINO (Qwen2.5) for privacy.
- [x] **Basic Agentic Tools**: Implement File Search, Web Scraper, and Python Sandbox.
- [x] **Vector Memory**: Setup SQLite-backed ChromaDB for conversation persistence.

---

## 🚀 Phase 2: Engagement & Ecosystem
*Growing the platform and making it highly extensible.*

- [ ] **Plugin Architecture**: Allow developers to write custom tools in Python/Node and hot-load them into AURA.
- [ ] **Multi-Agent Swarm Orchestration**: Integrate Hermes Agent for dynamic sub-agent delegation.
- [ ] **One-Click Deployments**: Fully containerized Docker Compose setups for team and enterprise self-hosting.
- [ ] **Cross-Platform Mobile Apps**: Polish and launch native iOS and Android versions of AURA.
- [ ] **Screen Content Awareness**: Enhance OCR and overlay capabilities to read and react to live screen elements securely.

---

## 🔮 Phase 3: Autonomous & Self-Improving
*Making AURA a true autonomous co-founder.*

- [ ] **Self-Coding Tools**: Enable AURA to write, test, and register its own Python tools on the fly based on user requests.
- [ ] **Advanced Long-Term Memory**: Implement a layered cognitive architecture (Fact Extraction -> Triplet Store -> Graph DB) to deeply understand user preferences over time.
- [ ] **Voice Interface**: Add real-time speech-to-speech interaction utilizing whisper and optimized TTS models.
- [ ] **Proactive Assistance**: Allow AURA to observe workflows and proactively suggest automations or fixes without explicit prompts.
